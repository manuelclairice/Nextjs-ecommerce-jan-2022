# NFT Market

NFT Market is a mockup E-commerce Web App where we can buy NFTs.

# Tech Stack
  ## Front/Backend
   - Next.Js
   - React
   - JavaScript
   - TYpeScript
   - @emotion
   - Js-cookie
   - Dotenv
   - Sharp

  ## Database
   - PostgreSQL
   - Ley
